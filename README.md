# Alphora Agent 101 - Frontend

Europe's First AI-Native MSP Support Platform - Client Portal

## Overview

A React-based dashboard for IT Managed Service Providers (MSPs) to create, view, and manage support tickets. The platform leverages AI-driven automation to handle L1/L2 support tickets, automating 60-70% of repetitive tasks.
